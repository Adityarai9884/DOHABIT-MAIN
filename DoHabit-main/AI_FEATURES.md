## Setup Instructions

### Step 1: Get Your Free Gemini API Key

1. Visit [Google AI Studio](https://makersuite.google.com/app/apikey)
2. Sign in with your Google account
3. Click "Get API Key" or "Create API Key"
4. Copy your API key (starts with "AIza...")

### Step 2: Configure AI in DoHabit

1. Open the app menu (â˜°)
2. Select "AI Settings"
3. Paste your API key in the input field
4. Enable AI Features toggle
5. Click "Save Settings"

### Step 3: Start Using AI Features

Once configured, AI insights will automatically appear:
- **Main Dashboard**: Daily motivation card at the top
- **Statistics View**: AI analysis of your habit progress
- **Automatic Updates**: Insights refresh as you track habits

## Privacy & Security

- **Local Storage**: Your API key is stored locally on your device only
- **No External Server**: Data is sent directly to Google's Gemini API
- **On-Demand**: AI features only activate when you use them
- **Optional**: You can disable AI features anytime in settings

- Manual refresh option available
